# Charlytaxi‑Air – HTML Spiele

Willkommen im offiziellen Spiele‑Hub von **Charlytaxi‑Air**.  
Hier findest du kleine, schnelle Browser‑Games, die direkt ohne Installation gespielt werden können.

## 🎮 Spiele

### 🌥️ Wolken‑Jagd  
Wolken abschiessen mit dem Cursor und klicken. Die Grossen mit 3 die mittleren mit 2 und die kleinen mit einem Trefferst.  
👉 https://simelb01-source.github.io/Wolken-Jagd/

### 🪨 Stein‑Hüpfer  
Mit der Leer Taste und der Pfeiltaste Laden und beim Loslassen springt der Frosch.  
👉 https://simelb01-source.github.io/stein-huepfer/

### 🕹️ Flipperspiel  
Ein klassisches Mini‑Flipper‑Game für zwischendurch.  
👉 https://simelb01-source.github.io/Flipperspiel/

---

## 📁 Inhalt des Repositories

- `index.html` – Startseite mit allen Spiel‑Links  
- Unterordner für jedes Spiel (falls vorhanden)  
- Ressourcen wie Bilder, Skripte oder Stylesheets

---

## 🚀 Nutzung

1. Repository klonen oder herunterladen  
2. `index.html` im Browser öffnen  
3. Spiel auswählen und direkt loslegen
